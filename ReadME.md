 # Task Description

 To implement an API with 2 endpoints:
 - that allows users to create a payment and persist in a datastore.
 - that returns the payment together with associated order.

# Implementation

## Technologies used
- C# .NET Core
- Postgres
- Docker
- ORM - Entity Framework
- Testing  - xunit, Postman
